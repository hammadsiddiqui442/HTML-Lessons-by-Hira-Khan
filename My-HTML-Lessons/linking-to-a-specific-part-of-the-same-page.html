The Cloud Native Computing program prepares the students to program in open source technologies like Linux, Docker, Kubernetes, Javascript, Node.js, React and React Native while interacting students to development of microservice and software development techniques on cloud platforms like AWS.









Blockchain Computing

The main emphasis of Blockchain program is to understand the blockchain like Etherium and HyperLedger (Fabric) and learn to program i language such as Solidity, JavaScript on frameworks like Node.js. The program also introduces them to developmentof Distributed Apps (DApps), frameworks like we3 and Traffic and in-depth understanding of Blockchan-as-a-service (BaaS) and how to develop and offer an Initial coin offering (ICO) of a crypto-token.









Internet of Things

The internet of Things program introduces the students to open source development technologies like Rust, Embeded Rust, WebAssembly, Hyper, Diesel, Google Assistant, and Kubernetes while making them efficient in developing microservice for IOT Edge, IOT Cloud, and Voice Computing for device control.
